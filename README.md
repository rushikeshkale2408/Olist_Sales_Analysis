# Olist_Sales_Analysis
Olist Sales Analysis 📊

📌 Domain

E-commerce / Retail Analytics

---

🧰 Tools Used

- SQL 
- Power BI (Dashboard Visualization)
- Tableau (Advanced Visualization)
- Excel (Data Preprocessing)
- 
---
📂 Dataset

This dataset is based on the Brazilian e-commerce platform Olist.
It contains information about orders, customers, products, sellers, payments, and reviews.

---

📊 Key KPIs

- Total Orders
- Total Revenue
- Average Order Value
- Delivery Time
- Cancellation Rate

---

📈 Analysis Performed

1. Sales Performance

- Monthly and yearly sales trends
- Revenue growth analysis

2. Customer Analysis

- Number of unique customers
- Customer purchase behavior

3. Product Analysis

- Top-selling categories
- High revenue-generating products

4. Delivery Analysis

- Average delivery time
- Late deliveries

5. Payment Analysis

- Most used payment methods
- Payment value trends

---
📊 Dashboard

Interactive dashboards created using:

- Power BI – KPI tracking and business insights
- Tableau – Advanced visual storytelling and trend analysis

---
📌 Key Insights

- Sales peak observed during festive seasons
- Certain product categories dominate revenue
- Faster deliveries improve customer satisfaction
- Majority of payments are done via credit card

---
🧠 Conclusion

This analysis helps businesses understand customer behavior, optimize delivery operations, and improve overall sales performance.
